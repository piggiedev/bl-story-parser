# BL Story Parser (Zero-Dep Edition)

A high-performance, regex-based scraper for the BL archive.
This project uses **native JavaScript** and **Bun** for maximum speed with zero external dependencies (no `node_modules` required!).

---

## 🛠 Installation

To run these scripts, you need **Bun** installed on your Mac. Bun is a fast all-in-one JavaScript runtime.

### 1. Install Bun

Open your terminal and run:

```bash
curl -fsSL [https://bun.sh/install](https://bun.sh/install) | bash
```
